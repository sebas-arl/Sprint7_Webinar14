## Car Sales Analysis: Mi Proyecto del Sprint 7
¡Hola! Este es el proyecto que armé para el Sprint 7. La idea principal fue trabajar con Herramientas de desarrollo de software, no enfocado en la limpieza, preparación y analisis de los datos, sino construyendo una herramienta que alguien más pueda usar.

# ¿Para qué sirve esta app?
Es una aplicación web interactiva que analiza un dataset de anuncios de venta de coches en EE. UU.

# ¿Qué hace?
- Checkboxes Interactivos: Tú decides qué quieres ver. Si quieres analizar el kilometraje, activas el histograma del Odómetro. Si te interesa ver cómo se deprecian los precios, activas el Gráfico de Dispersión.

- Visualización Dinámica: Puedes hacer zoom en los datos, descargar los gráficos o pasar el mouse para ver los detalles de cada coche (precio, año, etc.).

- Filtros Inteligentes: Limpié los datos (quitando precios exagerados o errores) para que lo que veas tenga mas sentido.
